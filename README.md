# Mama Fua Services

A responsive PHP and MySQL service booking platform for professional cleaning services.

## Features
- Premium landing page
- Service catalog and details pages
- Customer authentication and dashboard
- Booking flow and contact form
- Admin-ready database schema

## Setup
1. Create a MySQL database named `mama_fua_services`.
2. Import [database/schema.sql](database/schema.sql).
3. Update database credentials in [config.php](config.php).
4. Serve the project from a PHP-enabled web server such as XAMPP, WAMP, or Laragon.
5. Open [index.php](index.php) in the browser.
